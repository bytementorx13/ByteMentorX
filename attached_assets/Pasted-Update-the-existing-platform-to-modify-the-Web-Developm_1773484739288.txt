7. SECURE PAYMENT IMPLEMENTATION

Payments must be fake-proof.

Implement Razorpay security measures:

1. Razorpay Order Creation on Backend only.

2. Verify Payment Signature.

Example verification:

Use HMAC SHA256 with Razorpay secret key to verify:

order_id + "|" + payment_id

Compare generated signature with Razorpay signature.
